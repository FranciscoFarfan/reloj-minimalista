# Reloj Minimalista con Clima
Una página web minimalista que muestra la hora actual y el clima de tu ubicación.
Diseñada para ser simple, elegante y funcional.
Prueba en vivo <a href="https://franciscofarfan.github.io/Pruebas/reloj-minimalista/reloj.html"> Reloj Minimalista </a>

## Características
- Interfaz minimalista y limpia.
- Muestra la hora actual en tiempo real.
- Consulta y despliega el clima de tu ciudad.
- 100% desarrollado en HTML, CSS y JavaScript.
- Ligero y fácil de personalizar.
